Phase1 Functionality And Grammar Limitations:
  • halfwaytree will be a generalized symbolic execution engine
  • halfwaytree is limited to the following types of statements
    1. print
    2. assignment
    3. conditionals
  • halfwaytree is limited to the following types of conditionals
    1. >, <, =, !=
    2. if
    3. if, else
    4. if, elif
    5. non compound conditionals
      • ie: no and's nor or's
  • variables which are symbolic can be only strings and integers
  • all variables will be tracked symbolically
Symbolic Execution Steps Used In Phase1:
  1. Python will generate the abstract syntax tree (AST)
  2. Halfwaytree will parse the AST and generate a digraph of the source code
    • this can be represented graphically as well using Pygraphviz
  3. For each path, the code goes through and:
    • generates constraints
    • code checks constraint feasibility before it executes statement
    • symbolically executes code
      • uses exec to execute statements on nodes
      • updates symbolic variables
